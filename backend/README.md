# Backend

This is the backend service for the datodo application, which provides a platform for learning data structures through gamification.

## Description

The backend is built with NestJS and uses Prisma as the ORM to interact with a MySQL database. It provides RESTful APIs for managing data structures, quizzes, and user interactions.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Database

The application uses MySQL as its database. Make sure you have MySQL installed and running.

### Prisma

The database schema is managed using Prisma. To set up the database:

1. Create a `.env` file in the root directory with your database connection string:
```
DATABASE_URL="mysql://user:password@localhost:3306/datodo"
```

2. Run the migrations:
```bash
$ npx prisma migrate dev
```

3. Seed the database (optional):
```bash
$ npx prisma db seed
```

## API Documentation

The API documentation is available at `/api` when the server is running. It's built using Swagger UI.

## Authentication

The API uses JWT (JSON Web Tokens) for authentication. Protected routes require a valid JWT token in the Authorization header.
